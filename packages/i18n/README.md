# @sigx/i18n

Reactive localization for [SignalX](https://sigx.dev/core/) — namespaces, a master
locale with automatic fallback, API-defined targets (scopes), locale detection,
SSR-safe state transfer, and easy UI binding.

```sh
pnpm add @sigx/i18n
```

```ts
import { createI18n, useTranslation } from '@sigx/i18n';
```

Full documentation → **<https://sigx.dev/i18n/>**
